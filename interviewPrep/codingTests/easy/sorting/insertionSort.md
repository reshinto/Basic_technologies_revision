# Insertion Sort
Sample Input
```array = [8, 5, 2, 9, 5, 6, 3]```
Sample Output
```[2, 3, 5, 5, 6, 8, 9]```
```python
# solution 1
def insertionSort(array):
  # Write your code here.
  i = 0
  while i < len(array) - 1:
    for j in range(i + 1, -1, -1):
      if j > 0 and array[j] < array[j - 1]:
        array[j], array[j - 1] = array[j - 1], array[j]
    i += 1
  return array
```
